# Coding Assignment 12 - Web Component Library

Student Name: Arshpreet Kaur  
Project Name: kaur_arshpreet_ui_garden

## Description

This project is a React TypeScript UI component library created using Create React App, Storybook, and Styled Components.

## Components

- Button
- Label
- Text
- Table
- TableHeader
- TableRow
- TableCell
- TableFooter
- Dropdown
- RadioButton
- Img
- HeroImage
- Card

## Run Project

npm install

npm start

## Run Storybook

npm run storybook

Open:

http://localhost:6006

## Run Tests

npm test

## Build Production App

npm run build

## Docker Build

docker build -t kaur_arshpreet_coding_assignment12 .

## Docker Run

docker run --name kaur_arshpreet_coding_assignment12 -p 8083:8083 kaur_arshpreet_coding_assignment12

Open:

http://localhost:8083

## Notes

The application uses Styled Components for styling. Components include default and disabled states. Storybook controls are used to modify component properties.
